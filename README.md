# Synthetic_seis_APP
*Synthetic_seis_APP — программа для генерации синтетических сейсмограмм*

*Общее назначение программы*

**SeismogramGeneratorGUI** — это специализированное научное приложение для **генерации синтетических сейсмограмм** в формате SEG-Y. Программа создает моделированные сейсмические данные на основе заданных геологических границ и скоростей распространения волн.

**Зачем нужна программа:**

1. **Моделирование сейсмических данных** — создание синтетических записей для тестирования алгоритмов обработки
2. **Обучение и исследования** — генерация эталонных данных с известными параметрами для верификации методик
3. **Проектирование съемки** — оценка параметров полевых работ до их фактического проведения
4. **Анализ скоростных моделей** — сравнение отклика от эффективных и интервальных скоростей
5. **Подготовка тестовых наборов** — создание контролируемых данных для отладки ПО

---

*1. Вкладка «Основные параметры»*

- **Параметры сейсмограммы**:
  - Длина записи (мс)
  - Шаг дискретизации (мс)
  - Номер удара FFID
- **Тип скоростей**:
  - Только эффективные
  - Только интервальные
  - Оба типа
- **Флаг создания модели скоростей**

*2. Вкладка «Границы и скорости»*

- **Таблица границ**:
  - № границы, t0 (мс), Vэфф (м/с), Vинт (м/с)
  - Автоматический расчет интервальных скоростей по формуле Диксера
- **Редактирование**:
  - Добавление новой границы
  - Обновление параметров
  - Удаление выбранной
  - Очистка всех
  - Загрузка по умолчанию
- **Автонумерация** — автоматическая перенумерация при удалении/добавлении

*3. Вкладка «Геометрия»*

- **Параметры расстановки**:
  - Координата источника X (м)
  - Диапазон приемников: от, до, шаг
- **Интерактивное обновление** — перестроение геометрии

*4. Вкладка «Предпросмотр и генерация»*

- **Визуализация**:
  - Интерактивный график расстановки
  - Информация о количестве приемников и разносах
- **Управление файлами**:
  - Базовое имя файлов
  - Выбор папки сохранения (по умолчанию Рабочий стол)
  - Кнопка «Обзор...»
- **Лог выполнения** — подробный протокол всех операций

*5. Основные алгоритмы*

- **create_synthetic_seismogram** — генерация сейсмограммы с импульсами в точках прихода
- **effective_to_interval_velocities** — конвертация эффективных скоростей в интервальные
- **create_velocity_model** — создание модели интервальных скоростей во времени
- **save_segy_file** — экспорт в промышленный формат SEG-Y с полными заголовками

*6. Форматы выходных данных*

- `*_effective.sgy` — сейсмограмма с эффективными скоростями
- `*_interval.sgy` — сейсмограмма с интервальными скоростями
- `*_velocity_model.sgy` — модель интервальных скоростей (1 трасса)
